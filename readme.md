# Readme
### How to commit:
> 1. git add * (adds all changed files)  
> 2. git commit -m "message" (commits changes)  
> 3. git push origin master (pushes changes to origin remote in master branch)  

### How to pull:
> 1. git pull upstream master (Inn0's version)
